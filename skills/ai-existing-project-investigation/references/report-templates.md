## Evidence Rules

- Evidence is repository text, command output, tests, logs, schemas, or concrete runtime observations.
- Inference is a conclusion drawn from evidence.
- Do not write "confirmed" unless evidence directly supports it.
- If an important file was not read, list it instead of pretending coverage is complete.
- If cause is not proven, state the strongest hypothesis and what would prove or disprove it.
